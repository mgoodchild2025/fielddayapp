---
title: Discount Codes
description: Create promo codes to offer reduced registration fees.
---

<Info>
  Discount codes are available on the **Pro** and **Club** plans.
</Info>

## Create a discount code

1. Go to **Admin → Settings → Discount Codes**
2. Click **New discount code**
3. Fill in the details:

| Field | Description |
|---|---|
| **Code** | The text players enter at checkout (e.g. `SUMMER25`) |
| **Discount type** | Percentage off or fixed dollar amount |
| **Amount** | How much the code discounts (e.g. `25` for 25% or $25 off) |
| **Max uses** | Optional — limit how many times the code can be used |
| **Expiry date** | Optional — code won't work after this date |

4. Click **Save**

## How players use a code

During registration checkout, players see a **Discount code** field. They enter the code and the discounted price is applied before payment.

## View code usage

The Discount Codes page shows how many times each code has been used and the total discount given.

## Deactivate a code

Toggle a code to **Inactive** to stop it from being accepted without deleting it.

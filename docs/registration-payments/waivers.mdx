title: Waivers

## Create a waiver

1. Go to **Admin → Settings → Waivers**
2. Click **New waiver**
3. Give it a name (e.g. "Standard Liability Waiver 2026")
4. Write the waiver text using the rich text editor — you can add headings, bold text, bullet points, and links
5. Click **Save**

## Attach a waiver to an event

When creating or editing an event, select your waiver from the **Waiver** dropdown. Players will be required to read and sign it during the registration flow before they can complete checkout.

<Tip>
  Create a new waiver version each year so you have a clear record of which document each player signed and when. Don't edit an existing waiver that players have already signed — create a new version instead.
</Tip>

## View waiver signatures

Go to **Admin → Events → [Your event] → Registrations** and look for the waiver status column. A checkmark means the player signed the waiver. Click a player's name to see the exact timestamp of their signature.

## Waiver versions

Each saved waiver is versioned. When you update your waiver text, save it as a new version. This preserves a historical record showing exactly which version each player agreed to.

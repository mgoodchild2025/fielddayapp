title: Custom Navigation Links
description: Add up to 5 custom links to your public navigation bar.
---

## Overview

Add links to external URLs or uploaded PDF documents that appear in both the desktop nav and mobile menu of your public site. Useful for league policies, social media profiles, game video archives, contact pages, and more.

<Info>
  Custom navigation links require the **Starter plan** or higher.
</Info>

## Where to find it

**Admin → Settings → Navigation**

---

## Add a URL link

<Steps>
  <Step title="Click + Add Link">
    On the Navigation settings page, click **+ Add Link**.
  </Step>
  <Step title="Select URL">
    Choose **URL** as the link type.
  </Step>
  <Step title="Enter a label">
    Type the label that will appear in your navigation bar (e.g. "Contact Us").
  </Step>
  <Step title="Paste the URL">
    Enter the full URL. It must start with `https://`.
  </Step>
  <Step title="Choose tab behaviour">
    Toggle **Open in new tab** — this is on by default and recommended for external links.
  </Step>
  <Step title="Save">
    Click **Save**. The link appears in your public nav immediately.
  </Step>
</Steps>

---

## Add a document (PDF)

<Steps>
  <Step title="Click + Add Link">
    Click **+ Add Link** on the Navigation settings page.
  </Step>
  <Step title="Select Document">
    Choose **Document** as the link type.
  </Step>
  <Step title="Enter a label">
    Type the label that will appear in the nav (e.g. "League Rules").
  </Step>
  <Step title="Upload a PDF">
    Upload a PDF file (max 10 MB). The file is stored on Fieldday.
  </Step>
  <Step title="Save">
    Click **Save**. Document links always open in a new tab.
  </Step>
</Steps>

<Tip>
  Document links display a PDF icon in the mobile menu so players know they'll be opening a file.
</Tip>

---

## Reorder links

Use the **↑ ↓ arrows** on each link row to change the order they appear in the navigation bar. Changes take effect immediately.

---

## Edit or remove a link

- Click **Edit** on any link to update the label, URL, or replace the uploaded document.
- Click **Remove** to delete the link from your navigation.

---

## Limit

You can add a maximum of **5 custom links** per organization. When the limit is reached, the **+ Add Link** button is hidden until you remove an existing link.

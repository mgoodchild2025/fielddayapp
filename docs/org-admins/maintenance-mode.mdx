---
title: Maintenance Mode
description: Temporarily take your org's public site offline during updates or migrations.
---

## Overview

When maintenance mode is on, visitors to your public site see a "We'll be right back" page instead of your normal content. Org admins and platform admins can still access the admin panel normally during maintenance.

---

## Where to find it

Maintenance mode is managed by the **Fieldday team** via the Fieldday Super Console. Org admins cannot enable maintenance mode themselves.

<Info>
  Maintenance mode is managed by the Fieldday team. Email [hello@fielddayapp.ca](mailto:hello@fielddayapp.ca) to schedule a maintenance window.
</Info>

---

## What visitors see

While maintenance mode is active, visitors to your public site see a clean page with:

- Your org logo
- A wrench icon
- The heading **"We'll be right back"**
- An optional custom message (e.g. "We're migrating data and will be back in 30 minutes.")
- An optional **estimated return time** shown as "Expected back: [date/time]"

---

## Custom message

Platform admins can set a custom message that appears on the maintenance page — useful for letting players know what's happening and when to check back.

---

## Estimated return time

An optional date and time can be set alongside the maintenance message. It's shown on the maintenance page as an expected return time so visitors know when to check back.

---

## Platform admins bypass

Fieldday platform admins (employees) always see the normal site and are never shown the maintenance page, regardless of whether maintenance mode is active.

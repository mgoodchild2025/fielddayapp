title: Print Scoresheets
description: Generate print-ready score sheets and daily schedules for game day.

<Info>
  Print scoresheets are available on the **Pro** and **Club** plans.
</Info>

Fieldday can generate print-ready PDFs directly from your browser — no extra software needed.

## Print a daily schedule

The daily schedule is a one-page overview of all games on a given day, with blank score boxes for referees to fill in on the spot.

1. Go to **Admin → Events → [Your event] → Schedule**
2. Find the date heading you want to print
3. Click **Print Day** — a print-ready page opens in a new tab
4. Press **Ctrl+P** (or **Cmd+P** on Mac) and choose **Save as PDF** or send to your printer

## Print a game scoresheet

The game scoresheet is a detailed sheet for a single game, with score boxes for each set, period, or half (depending on the sport), and signature lines for referees and team captains.

1. In the schedule, find the game you want to print
2. Click the **🖨 Score** icon on the game row
3. A print-ready scoresheet opens in a new tab
4. Press **Ctrl+P** / **Cmd+P** to print or save as PDF

## Print a stat sheet

The stat sheet includes pre-filled player rosters for both teams, with columns for each stat tracked in your event, plus blank rows for walk-in players.

1. Find the game in the schedule
2. Click the **🖨 Stats** icon on the game row
3. Print or save as PDF

## Signature lines

Every scoresheet and stat sheet includes signature lines for:
- Referee 1
- Referee 2
- Home team captain
- Away team captain

These provide a physical record of the agreed result.

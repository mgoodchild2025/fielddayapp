<Info>
  Drop-in and recurring sessions are available on the **Pro** and **Club** plans.
</Info>

## Drop-in events

Drop-in events are single sessions that individual players can register for — no team required. Players pay a per-session fee at registration.

**Good for:** casual games, open gyms, one-off clinics, pickup nights.

### Create a drop-in event

1. Go to **Admin → Events → New event**
2. Select **Drop-in** as the event type
3. Set the session date, time, venue, and per-player price
4. Open registration

Players register individually and receive a confirmation email with the session details.

## Recurring sessions

Recurring sessions repeat on a set schedule — for example, every Tuesday night at 7pm. You create the session once and it generates individual instances on your chosen schedule.

**Good for:** weekly pickup games, training sessions, open gym nights.

### Create a recurring session

1. Go to **Admin → Events → New event**
2. Select **Recurring session** as the event type
3. Set the start date, frequency (weekly, biweekly), and end date
4. Set the per-session price and capacity

Each session instance appears on the schedule separately. Players can register for individual sessions or the whole series, depending on your settings.

## Check-in

For both drop-in and recurring sessions, use the **Check-in** feature on game day to mark which players attended. Go to **Admin → Events → [Session] → Check-in** and tap each player as they arrive.

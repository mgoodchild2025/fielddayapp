---
title: Set Up Registration
description: Configure fees, waivers, and registration settings for your event.
---

Registration settings are configured when you [create an event](/running-events/create-event) and can be updated at any time from the event's **Overview** tab.

## Open registration

When you're ready for players to sign up:

1. Go to **Admin → Events → [Your event]**
2. Click **Open registration** (or change the status dropdown to **Registration open**)

Your event will now appear on your public events page at `yourname.fielddayapp.ca/events`.

## Close registration

To stop accepting new registrations:

- Change the event status to **Active** — registration closes automatically
- Or manually set the **Registration closes** date when creating the event

## Registration modes

| Mode | How it works |
|---|---|
| **Team registration** | A team captain registers and pays for the whole team |
| **Individual registration** | Each player registers and pays separately, then joins or creates a team |

## Capacity limits

Set **Max teams** or **Max participants** to automatically close registration when the event is full. Players will see an "Event full" message if they try to register after capacity is reached.

<Tip>
  Leave capacity blank if you don't want a limit. You can always close registration manually.
</Tip>

## Waitlist

When an event reaches capacity, late registrations are placed on a waitlist. If a spot opens up, you can manually promote waitlisted players to active from the Registrations tab.

## View registrations

Go to **Admin → Events → [Your event] → Registrations** to see everyone who has signed up, their payment status, and waiver signature status.

## Event types

| Type | Best for |
|---|---|
| **League** | Regular season play with standings — teams play each other over multiple weeks |
| **Tournament** | Single or double-elimination bracket play, usually over one or two days |
| **Drop-in** | One-off sessions players can join individually, with per-session pricing |
| **Recurring session** | Repeated sessions on a set schedule (e.g. every Tuesday night) |

## Create a league or tournament

1. Go to **Admin → Events**
2. Click **New event**
3. Fill in the event details:

### Basic info
- **Name** — what players will see (e.g. "Summer Volleyball League 2026")
- **Sport** — used to set default scoring rules and stat tracking
- **Event type** — League, Tournament, Drop-in, or Recurring session

### Dates
- **Season start / end** — the date range the event runs
- **Registration opens / closes** — when players can register

### Venue
- **Venue name and address** — shown on player-facing schedules
- **Surface type** — indoor, outdoor, or both

### Registration settings
- **Max teams / participants** — leave blank for unlimited
- **Team size** — min and max players per team
- **Join policy** — Open (anyone can join a team), Captain invite, or Admin only

### Fees
- **Price** — per player or per team fee in dollars
- **Payment mode** — per player or per team

### Rules
Optionally attach a rules document. You can write custom rules or select from a saved template.

### Waivers
Attach a liability waiver that players must sign during registration.

## Save and publish

Click **Create event**. The event is created in **Draft** status — players can't see it yet.

When you're ready to open registration, change the status to **Registration open** from the event overview page.

## Event statuses

| Status | What it means |
|---|---|
| **Draft** | Not visible to players, still being set up |
| **Registration open** | Players can register and pay |
| **Active** | Registration closed, season is underway |
| **Completed** | Season finished, standings locked |
| **Archived** | Hidden from the public events list |
